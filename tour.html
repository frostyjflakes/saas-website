Basic Features

Artboard Tool

Pencil Tool

Shape Tool

A tool that allows you to make geometric shapes

A tool that allows you to pick, mix and create a variety of colours

A great tool for making posters and other projects with type

A paint bucket that covers more ground than the Brush Tool

A faster, more accurate way to duplicate your shapes
